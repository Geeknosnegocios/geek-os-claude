Analise um arquivo de dados e gere um resumo executivo com insights acionáveis.

### Uso:

```
/analisar-dados dados/seu-arquivo.csv
```

Se nenhum arquivo for especificado, liste o que existe em `dados/` e peça para escolher.

### Execute a análise:

1. Leia o arquivo (CSV, XLSX, TXT ou PDF)
2. Identifique o que os dados representam
3. Mapeie as colunas/campos principais
4. Calcule as métricas mais relevantes (totais, médias, variações, top/bottom)
5. Identifique padrões, anomalias e tendências

### Entregue um relatório com:

**Resumo executivo** (3–5 frases sobre o que os dados mostram)

**Principais números:**
- Os 5 indicadores mais importantes
- Comparações relevantes (período, categoria, produto, canal)

**Insights:**
- O que está indo bem
- O que está preocupante
- O que foi inesperado

**Recomendações:**
- 3 ações concretas baseadas nos dados, ordenadas por impacto

Salve o relatório em `dados/relatorios/analise-[nome-arquivo]-[data].md`.
