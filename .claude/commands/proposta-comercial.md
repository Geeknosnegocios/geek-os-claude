Gere uma proposta comercial profissional em HTML a partir de um briefing.

### Colete o briefing perguntando:

1. Nome do cliente / empresa para quem é a proposta
2. Qual produto ou serviço está sendo proposto?
3. Qual é o valor (ou faixas de valor)?
4. Quais são os entregáveis — o que o cliente recebe exatamente?
5. Qual é o prazo de entrega ou duração do serviço?
6. Tem alguma condição especial? (desconto, parcelamento, bônus?)
7. Qual é o problema que o cliente tem e que essa proposta resolve?

### Leia o contexto de marca:

Verifique `marca/` para cores, nome da empresa, logo (se houver caminho) e tom de voz.

### Gere o HTML completo com:

- Cabeçalho com nome da empresa e data
- Seção de apresentação do problema do cliente
- Descrição da solução proposta
- Lista de entregáveis detalhada
- Investimento com forma de pagamento
- Prazo e próximos passos
- Seção de garantia (se aplicável)
- Rodapé com dados de contato
- Estilo visual limpo, profissional, compatível com a identidade da marca

### Salve o arquivo em:

```
operacoes/propostas/proposta-[nome-cliente]-[data].html
```

Informe o caminho do arquivo e diga como abrir no navegador.
