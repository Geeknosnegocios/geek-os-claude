Crie um slide ou card visual profissional em HTML para apresentação ou post.

### Pergunte:

1. Qual é o conteúdo do slide? (título, texto, dados que devem aparecer)
2. Para que vai usar? (apresentação, story, card de post, capa de vídeo?)
3. Formato: horizontal (16:9), vertical (9:16) ou quadrado (1:1)?

### Leia `marca/` para pegar identidade visual.

### Gere um arquivo HTML com:

- Layout limpo e impactante
- Tipografia grande e legível
- Cores da marca
- O conteúdo solicitado bem diagramado
- Sem bordas ou elementos desnecessários — visual direto

### Salve em:

```
conteudo/slides/slide-[tema]-[data].html
```

Informe como abrir no navegador e como fazer print/screenshot para usar como imagem.
