Crie uma pasta de projeto novo com CLAUDE.md dedicado, após uma entrevista sobre o projeto.

### Conduza a entrevista perguntando:

1. Qual é o nome do projeto?
2. O que é esse projeto? (produto, campanha, cliente, evento, lançamento?)
3. Qual é o objetivo principal? O que define o sucesso desse projeto?
4. Qual é o prazo ou data de entrega?
5. Quem está envolvido? (só você, equipe, cliente externo?)
6. Quais são os entregáveis — o que precisa ser produzido?
7. Tem algum contexto importante que eu preciso saber antes de começar?

### Crie a estrutura:

```
operacoes/projetos/[nome-do-projeto]/
├── CLAUDE.md          → contexto específico do projeto
├── briefing.md        → resumo da entrevista
├── tarefas.md         → lista de tarefas e status
└── arquivos/          → pasta para arquivos do projeto
```

### O CLAUDE.md do projeto deve conter:

- O que é o projeto e seu objetivo
- Prazo e marcos importantes
- Entregáveis esperados
- Pessoas envolvidas e seus papéis
- Contexto e restrições importantes
- Instrução para o Claude se comportar como gestor deste projeto específico

Ao criar, informe o caminho da pasta e sugira a primeira ação para começar o projeto.
