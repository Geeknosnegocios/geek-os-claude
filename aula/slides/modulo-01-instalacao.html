MOD_01 · 1.1 VS CODE
Aula 1.1 — VS Code

Seu cockpit de comando.

1. Acesse: code.visualstudio.com
2. Download for Windows (ou Mac)
3. Instale normalmente → next → finish
// verifique:
$ code --version
1.89.0
03
MOD_01 · 1.2 NODE.JS
Aula 1.2 — Node.js

O motor por baixo de tudo.

1. Acesse: nodejs.org
2. Baixe a versão LTS (recomendada)
3. Instale normalmente
// verifique no terminal do VS Code:
$ node --version
v20.11.0 ← qualquer número aqui = ok
04
MOD_01 · 1.3 CLAUDE CODE
Aula 1.3 — Claude Code

Instalando o sócio.

// instale globalmente via npm:
$ npm install -g @anthropic-ai/claude-code
// configure a chave da API Anthropic:
→ console.anthropic.com → API Keys → Create Key
// verifique:
$ claude --version
claude/1.x.x ← sócio instalado ✓
